S3 server access logs can now be delivered directly to CloudWatch Logs in structured format, where they parse automatically and become queryable within seconds. Optional S3 Tables integration mirrors logs as Apache Iceberg data for SQL analytics, eliminating ETL setup and Glue crawler maintenance.
